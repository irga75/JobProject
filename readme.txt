Проект Jooble - сайт поиска работы.

1) Есть вакансии
2) Есть пользователи
3) Пользователь может стать работодателем
4) Пользователь может создать своё резюме
5) Работодатель может создавать вакансии

Схема сайта

Главная страница:
- Список вакансий
- Кнопка зарегестрироваться и войти
- Если пользователь уже зашёл в аккаунт, появляется кнопка "зайти в личный кабинет"

Личный кабинет:
- Просмотр данных пользователя
- Кнопка создать резюмэ или редактировать резюме если резюме уже есть
- Кнопка выйти из аккаунта
